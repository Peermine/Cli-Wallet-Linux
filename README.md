
## Linux & Ubuntu Wallet-Cli

### Download Wallet-Cli

    wget http://peermine.com/index/linux-wallet.zip
    unzip linux-wallet.zip wallet
    cd wallet

### Open Permission

    chmod u+x walletd
    chmod u+x miner
    chmod u+x peermined
    chmod u+x simplewallet
    
### Run Daemon

    ./peermined
    
### Create Wallet

    ./simplewallet
